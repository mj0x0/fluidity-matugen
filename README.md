# Fluidity - An accordion based startpage
This is my fork of the great Fluidity startpage. Main changes are matugen support, clamp() instead of fixed height for better responsviness, and some new images. 
Use `input_template.j2` in your matugen config.toml, and make sure to output theme.css into public.
